Let's learn how the browser understands the type of data storage

Java Script executes code in two phases.. memory creation phase and execution phase.
There are 3 ways that we store the data "VAR, LET, CONST".
Let's understand one-by-one.

console.log(a) --gives output as undefined
var a=10
console.log(a) --gives 10 as output

console.log(b) --Throws error
let b=20
console.log(b)
